Your blog has been migrated to Sanity (free, industry-standard CMS).
Follow these 4 steps to go live:

───────────────────────────────────────────────────────────────
STEP 1: Create Sanity Account (2 minutes)
───────────────────────────────────────────────────────────────

1. Visit: https://www.sanity.io
2. Click "Get started for free"
3. Sign up with GitHub, Google, or Email
4. Create new project: "California Dog Parks Blog"
5. Copy your Project ID (looks like: abc12xyz)

───────────────────────────────────────────────────────────────
STEP 2: Update .env.local (30 seconds)
───────────────────────────────────────────────────────────────

Open: .env.local
Find: NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id_here
Replace: your_project_id_here with your actual Project ID

───────────────────────────────────────────────────────────────
STEP 3: Deploy Schemas (2 minutes)
───────────────────────────────────────────────────────────────

Run these commands (replace YOUR_PROJECT_ID):

npm install -g @sanity/cli

npx sanity@latest init \
  --project YOUR_PROJECT_ID \
  --dataset production \
  --output-path sanity-studio

cd sanity-studio
npx sanity deploy

(Choose a hostname like: californiadogparks)

───────────────────────────────────────────────────────────────
STEP 4: Add Content & Test (1 minute)
───────────────────────────────────────────────────────────────

1. Open: https://your-hostname.sanity.studio
2. Create an Author (your name)
3. Create a Category (e.g., "Park Reviews")
4. Create a Blog Post and click "Publish"

Then test:

npm run dev

Visit: http://localhost:3000/blog
